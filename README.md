# C Programming Basics

This repository contains basic C programs written while learning the fundamentals of programming.

The programs cover important concepts such as:

- Input and Output
- Conditional Statements
- Loops
- Functions
- Arrays
- Basic problem solving using C

These programs help build a strong foundation in programming logic and structured programming.

## Programs Included

1. Hello World
2. Even or Odd Number
3. Factorial of a Number
4. Prime Number Check
5. Palindrome Number Check
6. Fibonacci Series
7. Reverse a Number
8. Sum of Digits
9. Multiplication Table
10. Largest Element in an Array
11. Sum of Array Elements
12. Simple Calculator
